## About ChatApp

ChatApp is a modern, collaborative real-time chat platform built with Laravel and Alpine.js. Designed for teams and communities, it supports group rooms, direct messaging, invitations, and role-based access. The application emphasizes security, scalability, and a responsive user experience.

### Features

- Real-time group and direct messaging
- Room creation and invitations
- Role-based access (Admin, User)
- Responsive UI with Alpine.js and Tailwind CSS
- Secure authentication and authorization

### Technologies Used

- **Laravel** (PHP): Backend framework for routing, models, controllers, policies, and services
- **Alpine.js**: Lightweight JavaScript framework for interactive UI components
- **Tailwind CSS**: Utility-first CSS framework for responsive design
- **Vite**: Modern build tool for asset bundling and hot module replacement
- **SQLite**: Relational database for persistent storage
